PROJECT OVERVIEW
------------------------------------------------------------
This assignment focuses on automating the testing process of a
Singlish to Sinhala transliteration system using Python-based
UI automation.

The automation framework validates transliteration outputs
against predefined expected results stored in an Excel sheet.

------------------------------------------------------------
EXECUTION STEPS
------------------------------------------------------------
1. Open terminal in project folder

2. Activate virtual environment (optional)
   venv\Scripts\activate

3. Install required packages
   pip install -r requirements.txt
   playwright install

4. Execute automation
   python test_automation.py --excel "IT - Test cases.xlsx" --url "https://www.pixelssuite.com/chat-translator"

VALIDATION METHOD
------------------------------------------------------------
Strict output comparison is applied.

A test case is marked as FAIL if there are:
- Spelling mismatches
- Formatting differences
- Spacing inconsistencies
